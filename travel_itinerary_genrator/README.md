# Conversational Travel Itinerary Generator

This project is a conversational AI assistant that generates customized travel itineraries based on user constraints such as budget, destination, number of days, and number of people.

## Features
- Conversational chat interface
- Personalized itinerary
- Budget-based planning
- Day-wise schedule generation

## Technologies Used
- Python
- OpenAI API
- Prompt Engineering
- Google Colab
