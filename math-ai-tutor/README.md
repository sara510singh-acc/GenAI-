# AI Math Tutor – Step-by-Step Learning Assistant

## Project Description
The AI Math Tutor is a conversational AI assistant designed to help school students solve math problems step-by-step without directly giving the final answer. The AI provides hints, guiding questions, and feedback to help students learn the problem-solving process.

This project uses Generative AI and Prompt Engineering to create an educational assistant.

---

## Features
- Helps students solve math problems step-by-step
- Does not reveal the final answer directly
- Provides hints and guidance
- Interactive conversational chat
- Supports algebra and basic math problems

---

## Technologies Used
- Python
- OpenAI API
- Prompt Engineering
- Google Colab
- GitHub

---

## How It Works
The system uses a system prompt that instructs the AI to behave like a tutor. Instead of giving direct answers, the AI asks guiding questions and provides hints so that the student can solve the problem on their own.
